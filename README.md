Exemples d'utilisation de la lib `react-query`

Inspiration du projet https://github.com/chawax/react-query-demo + utilisation de l'API publique SWAPI

- lancement db: `yarn db`
- lancement front: `yarn start`

les routes utilisables sont indiquées dans le fichier `config/router.js`

hooks donnés à titre d'exemple:
- useQuery
- useInfiniteQuery
- useMutation
